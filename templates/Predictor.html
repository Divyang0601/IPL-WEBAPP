Welcome to the Real-time IPL Score Predictor WebApp

This is a web application that can be used to predict the first inning score of an IPL match, This application utilises the complete IPL match records (till 2020) and uses ML Ridge Regression algorithm to predict the score

Technology and Framework used -
  • ML Ridge Regression Algorithm to predict the estimated score
  • Python libraries like numpy, pandas and seaborn for analysis of the dataset
  • Python's Flask Framework
  • HTML, CSS, Bootstrap and Jquery to design interactive website
  • SQL Database linked with Flask to collect feedback from viewers
  • Heroku Web Hosting service to deploy the Web Application
Visit the predict page to use the model and feel free to play around, Your feedback/suggestion in the forum would be highly appreciated
